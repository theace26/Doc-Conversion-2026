Filter Bar + Table

Category toggle buttons across the top (like chips), search box, then a full-width table below. Clicking a row expands an inline detail panel. Action icons on each row for Viewer and Browse.

Sidebar Categories + Table

Left sidebar with category list (counts as badges). Clicking a category filters the main table. Detail panel slides in from the right when a row is clicked.
