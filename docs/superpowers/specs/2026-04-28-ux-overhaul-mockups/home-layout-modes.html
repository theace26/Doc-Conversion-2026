4. Settings → Display → Home layout — visual three-up picker
MarkFlow v0.34.1-dev
SearchPipelineConvertSettings
All settings/Display

Display.

How content shows up — including how busy your home page should feel.

Home page layout
Pick the home that fits your day. You can change this anytime; the keyboard shortcut ⌘\ cycles between the three.
Maximal
Search + browse rows.
Recent
Search + history only.
Minimal
Just the search box.
Track recent searches
Save your search query history so it shows up on the Recent layout. Off → nothing is logged; the Recent layout will show "Recently opened" only.

React

✓

Three modes feels right

Maximal / Recent / Minimal covers the spectrum. Lock it.

✓

Add the ⌘\ keyboard cycle

One shortcut cycles through all three layouts — same key any time you want to swap (e.g., quick switch from Maximal to Minimal when sharing your screen in a meeting).

Q

Add a layout quick-toggle in nav

A small icon next to the avatar lets you swap layouts without going into Settings. Useful if you cycle often.

P

Default "Track recent searches" to OFF for privacy

Currently mocked as ON. For an internal IBEW tool with shared workstations, OFF-by-default may be more appropriate.

D

Pick the system default mode

Currently the mockup defaults to Maximal. Should new users land on Maximal (welcoming, demonstrates capability), Recent (functional immediately), or Minimal (calm)? Each has a case.

In terminal: which default mode (Maximal / Recent / Minimal) for new users, and whether the ⌘\ cycle + a nav quick-toggle are both worth it or one is enough.
