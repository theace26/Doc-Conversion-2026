4. Locked decisions — nav + avatar menu

Top nav (final)

  • Member role: Logo · Search · Convert · Layout-icon · Avatar
  • Admin / operator role: Logo · Search · Activity · Convert · Layout-icon · Avatar
  • Settings link — removed from nav. Lives entirely behind the avatar.
  • Activity — admin/operator only. Members never see it.

Avatar menu (final)

  • Header — avatar, name, role pill, scope ("IBEW Local 46").
  • Personal (everyone) — Profile, Display, Pinned folders, Notifications. API keys only visible to admin/operator.
  • System (admin only) — Storage, Pipeline & lifecycle, AI providers, Account & auth, Database health, Log management. Section header has "Admin only" gate badge.
  • "All settings →" — jumps to the card-grid overview for rarer deep dives.
  • Help — Help & docs, Keyboard shortcuts, Report a bug.
  • Sign out — danger color.
  • Build info — bottom strip, monospace, dev-only.
