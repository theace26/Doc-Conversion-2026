Which is the product you'd want to live in?

A

Premium — reduced density

Apple did the ops tool. Hero on every page. Most data hides behind progressive disclosure. Fewer pages survive.

B

Hybrid — chrome + density

Same headline / nav / type / color / buttons as A, but interiors keep working at today's density. Most pages survive.

AB

Mix — A for outer pages, B for ops pages

Convert + Settings get the premium A treatment. Pipeline / Bulk Jobs / Batch Mgmt / Logs stay B-style because operators actually need the density there.

All accent colors, type choices, button shapes are placeholders. Lock the level of transformation first; iterate on the visual details after.
