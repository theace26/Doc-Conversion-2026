← All settings

Display.

How content shows up across MarkFlow.

Default content density
Used on Search, expanded sections, and Batch Management. The on-page toggle overrides this for the current view.
Snippet length
How much content to preview inside each card. Longer means richer browsing, slower render.
Show file thumbnails
For PDFs and image-bearing formats, render the first page as the card thumb instead of the markdown snippet.

React

✓

Snippet thumbs are right

Document-paper aesthetic with serif body text reads as "this is a document tool." Keep.

✓

Three density modes are right

Cards / Compact / List covers the spectrum. Persists from Settings, on-page toggle overrides per view.

P

Render first PDF page for PDF cards

For PDFs and image-bearing formats, prefer the actual page-render over the markdown snippet (toggle in Settings — off by default for performance, on for visual users).
